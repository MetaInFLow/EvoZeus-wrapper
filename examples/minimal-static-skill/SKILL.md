# Minimal Static Skill

Use this example when demonstrating how EvoZeus-wrapper records a static Skill run.

## Behavior

1. Restate the user's task in one sentence.
2. List assumptions explicitly.
3. Provide the smallest useful next action.

## Stop Conditions

Stop and ask when the task target is ambiguous or the requested output could expose private data.
